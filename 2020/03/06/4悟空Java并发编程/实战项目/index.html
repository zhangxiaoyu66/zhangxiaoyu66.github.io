高性能缓存

缓存是在实际生产中非常常用的工具，使用缓存可以避免重复计算，提高吞吐量

最初级的缓存可以用一个Map来实现，但一个功能完备，性能强劲的缓存，需要考虑的点就非常多了，从最简单的HashMap入手，一点点提高项目缓存的性能

可以在此情况下进行升级

不足

性能待优化
